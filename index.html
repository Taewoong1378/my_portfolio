"코딩을 즐기는 인재"

저는 코딩을 하는 것이 너무 즐겁습니다. 어렸을 적부터 여러가지 아이디어를 생각하고 구현해내는 것에 큰 흥미를 느꼈습니다. 웹 프로그래밍은 제가 상상만 해왔던 것들을 웹이라는 하나의 도화지에 그려낼 수 있게끔 해줍니다. 프로그래밍을 공부하면 공부할수록 제 아이디어를 구현할 수 있는 범위가 넓어지는 것이 너무 재밌고 설렙니다. 스마트동스쿨에서 구상한 여러 아이디어들을 실현해나가고 싶습니다.

"협업을 즐기는 인재"

저는 사람과 함께하는 것을 매우 좋아합니다. 이러한 성향을 바탕으로 고려대학교 중앙 컴퓨터 동아리 KUCC에서 개발 분야에 있는 동기 및 선후배들과 함께 활동하며 협업하였습니다. 협업을 위한 도구인 Slack, github, gater.town 등을 자유롭게 활용할 수 있습니다. 스마트동스쿨의 개발팀과 함께 협업할 생각에 벌써부터 설렘이 가득합니다.

HTML/CSS :
HTML/CSS의 경우, 태그 활용에 익숙하며 반응형 웹을 자유자재로 구현할 수 있습니다. 또, CSS 부트스트랩과 ant design 등을 활용해 빠른 결과물을 창출할 수 있습니다.

JAVASCRIPT & REACT :
JS의 경우, JQUERY를 활용할 순 있으나 DOM을 활용한 바닐라 JS 코드 작성을 더 선호합니다. 또, React.js와 Redux, Next.js를 조합하여 필요한 UI를 구현해낼 수 있으며 실제 서비스를 제작해본 경험이 있습니다.

NODE.JS & MYSQL :
Node.js의 경우, React와 함께 활용해 SNS 서비스 제작, 쇼핑몰 사이트 제작, 유튜브 사이트 제작 프로젝트들을 진행했습니다. MYSQL의 경우에는 SQLD 자격증을 취득하며 SQL과 RDBMS에 대한 기본 지식과 쿼리문에 대한 지식을 습득하였고, 해커랭크, 리트코드의 문제들을 다수 풀며 쿼리 활용 능력을 길렀습니다.

PHP :
PHP의 경우, 러닝 PHP(데이비드 스클라)의 내용을 처음부터 끝까지 숙지하며 기본적인 PHP의 문법부터 MYSQL과의 연동, 디버깅과 테스팅, 라라벨 프레임워크 등에 대한 지식을 습득하였습니다.

JAVA :
Java의 경우, 기본부터 고급 과정까지 문법 공부를 완료하고 OCAJP 자격증을 취득하며 Spring, JSP 등 회사에서 요구하는 추가적인 기술 스택들을 빠르게 습득할 수 있는 준비가 완료돼있습니다.
