HTML/CSS :
HTML/CSS의 경우, HTML5 태그 활용에 익숙하며 반응형 웹을 자유롭게 구현할 수 있습니다. 또, CSS 부트스트랩과 Ant design 등을 활용해 빠른 결과물을 창출할 수 있습니다.

JAVASCRIPT & REACT :
JS의 경우, JQUERY를 활용할 순 있으나 DOM을 활용한 바닐라 JS 코드 작성을 선호합니다. 또, React.js와 Redux, Next.js를 조합하여 더미데이터를 생성하고 필요한 UI를 구현해낼 수 있으며 실제 서비스를 제작해본 경험이 있습니다.

NODE.JS & MYSQL :
Node.js의 경우, React와 함께 활용해 SNS 서비스 제작, 쇼핑몰 사이트 제작 프로젝트 등을 진행한 경험이 있습니다. MYSQL의 경우에는 SQLD 자격증을 취득하며 SQL과 RDBMS에 대한 기본 지식과 쿼리문에 대한 지식을 습득하였고, 해커랭크, 리트코드의 문제들을 다수 풀며 쿼리 활용 능력을 길렀습니다.

PHP :
PHP의 경우, 러닝 PHP(데이비드 스클라)의 내용을 처음부터 끝까지 숙지하며 기본적인 PHP의 문법부터 MYSQL과의 연동, 디버깅과 테스팅, 라라벨 프레임워크 등에 대한 지식을 습득하였습니다.

JAVA :
Java의 경우, 기본부터 고급 과정까지 문법 공부를 완료하고 OCAJP 자격증을 취득하며 Spring, JSP 등 회사에서 요구하는 추가적인 기술 스택들을 빠르게 습득할 수 있는 준비가 완료돼있습니다.

개발자는 흔히 앉아서 작업만 하는 정적인 직업으로 분류되며 실제로 그런 개발자들이 많습니다. 또, 훌륭한 프로그램을 개발했음에도 불구하고 적절한 PR을 버무리지 못해 상품의 가치를 끌어올리지 못하는 경우도 많습니다. 저는 그런 개발자가 아닌, 자기 PR이 가능한 개발자가 되고 싶습니다. 남들 앞에서 제 생각을 정리해 말하는 것을 좋아하기에 단순히 프로그램을 제작하는 것을 넘어 적절한 PR로 스스로 제 프로그램의 가치를 높이는 스티브 잡스 같은 개발자가 되고 싶습니다.

더 나아가 도전을 두려워 않는 개발자가 되고 싶습니다. 갑작스럽게 통계에서 개발 분야로 진로를 틀고 산업기능요원 도전을 이유로 휴학을 결심했을 때, 모두가 왜 굳이 힘든 길을 가냐며 만류했습니다. 하지만 저는 도전이란 실패 여부와 상관없이 항상 큰 가치가 있다고 생각하기에 이번 도전 또한 저에게 좋은 밑거름이 되리라 믿습니다. 앞으로도 지금처럼 끝없이 도전하며 그저 그런 개발자가 아닌, 국내에서 손 꼽히는 최고의 개발자가 되는 것이 제 목표입니다.
